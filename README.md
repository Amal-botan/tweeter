# Tweeter Project

Tweeter is a simple, single-page Twitter clone. This project was created to practice HTML, CSS, JS, jQuery and AJAX front-end skills, and Node, Express back-end skills.

## Dependencies

- Express
- Node 5.10.x or above
- Timeago.js
- Body-parser
- Chance
